# DevPulse — Dark Editorial Tech Blog

Blog técnico com design minimalista e moderno, construído com vanilla HTML/CSS/JS.

**Features:** busca em tempo real, filtro por categorias, 6 artigos completos, dark theme editorial, fully responsive, zero dependências.

**Stack:** HTML5 • CSS3 • JavaScript ES6+ • Google Fonts

## Features

✨ **Design Dark & Editorial** — Inspirado em Hashnode/dev.to com paleta de cores minimalista e tipografia refinada  
🔍 **Busca em tempo real** — Busca client-side indexada de posts e tags  
🏷️ **Filtro por categoria** — Categorias funcionais: DevOps, Back-end, Front-end, IA, Segurança  
📖 **Artigos completos** — 6 posts individuais com TOC sticky, reading progress bar e author cards  
🎨 **Fully responsive** — Mobile-first, sem frameworks CSS  
⚡ **Zero dependências** — Apenas HTML, CSS e vanilla JS  

## Stack

- HTML5 semântico
- CSS3 (custom properties, grid, flexbox)
- JavaScript ES6+ (scroll reveal, search, filtering)
- Google Fonts (Fraunces, DM Sans, JetBrains Mono)
